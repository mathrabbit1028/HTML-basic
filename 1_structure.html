HTML의 구조

HTML에는 거의 대부분의 HTML 문서에 사용되는 태그가 있습니다. 이러한 태그를 알아보도록 합시다.

html 태그

이 태그는 안의 내용이 HTML 코드임을 나타냅니다. 즉, 모든 태그는 html 태그 내부에 위치해야 합니다.
사용: <html>......</html>

head 태그

이 태그는 안의 내용이 문서의 전체적인 정보임을 나타냅니다.
사용: <head>......</head>

title 태그

이 태그는 웝페이지의 제목을 부여합니다. head 태그 내부에 위치해야 합니다.
사용: <title>......</title>

meta 태그

이 태그는 인코딩을 설정할 수 있습니다. head 태그 내부에 위치해야 합니다.
사용: <meta charset = "......">
대부분 utf-8의 인코딩을 사용하므로 <meta charset = "utf-8">로 적으면 됩니다.

body 태그

이 태그는 안의 내용이 문서의 본문임을 나타냅니다.
사용: <body>......</body>

목차 태그

모든 페이지와 글에는 목차가 있어야 합니다. 목차를 만드는 것과 같이 내용을 나열하는 태그를 알아보도록 합시다.

li 태그

이 태그는 여러 개의 내용을 나열할 때 각각의 내용을 구분합니다. 즉, 이 태그 안에 하나의 내용이 포함됩니다.
즉, 여러 개의 내용을 나열하려면 아래 예시를 여러 줄 적어야 합니다.
사용: <li>......</li>
li 태그는 아래에서 설명할 ol 태그와 ul 태그 내부에 있어야 합니다. 만약 그렇지 않을 경우 글머리 기호는 점으로 설정됩니다.

ol 태그

이 태그는 목차에 순서가 있음을 나타냅니다(ordered list). 즉, 이 태그 안에 있는 내용(li 태그)는 글머리 기호가 숫자로 표시됩니다.
사용: <ol>......</ol>

ul 태그

이 태그는 목차에 순서가 없음을 나타냅니다(unordered list). 즉, 이 태그 안에 있는 내용(li 태그)는 글머리 기호가 점으로 표시됩니다.
사용: <ul>......</ul>

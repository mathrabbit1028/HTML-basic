이미지 넣기

웹페이지에서 이미지는 생명과도 같습니다. 이미지가 있어야 웹페이지가 풍성해지기 때문이죠. 이미지를 넣는 태그를 알아보도록 합시다.

이미지를 넣는 태그는 img 태그입니다.
사용: <img src="......" width="......">
img 태그는 앞의 태그와 다르게 뒤에 다른 요소가 필요로 합니다. <img>만으로는 모든 정보가 주어지지 않기 때문입니다.
이렇게 태그 뒤에 오는 추가 정보를 속성(Attribute)라 합니다.

src 속성

src는 이미지의 주소를 나타냅니다. 같은 폴더 안에 있을 경우에는 파일 이름만 적으면 됩니다.
자세한 내용은 HTML 상대 경로를 참고하세요.

width 속성

width는 이미지 파일의 너비를 나타냅니다. 단위는 픽셀입니다.
화면 크기에 대한 비율로 나타내고 싶다면 % 단위를 사용하여 100%와 같이 나타내면 됩니다.
